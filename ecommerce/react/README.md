# AG Grid Ecommerce Example - React + TypeScript

This is a React + TypeScript + Vite implementation of an ecommerce demo using AG Grid.

## Development

```bash
npm install
npm run dev
```

The dev server will start on `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
